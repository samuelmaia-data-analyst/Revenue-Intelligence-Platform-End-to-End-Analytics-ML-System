# dbt Layer

This dbt project sits on top of the local SQLite warehouse generated by the pipeline.

## Purpose

- formalize analytics-ready models on top of warehouse tables
- expose semantic revenue metrics for finance-grade governance
- add dbt-native tests on the curated layer

## Expected warehouse

Run the pipeline first so the SQLite database exists at:

- `data/warehouse/revenue_intelligence.db`

## Example profile

Copy `dbt/profiles/profiles.yml.example` into your local dbt profiles directory and adjust paths if needed.

## Typical commands

```powershell
dbt --project-dir dbt debug
dbt --project-dir dbt run
dbt --project-dir dbt test
dbt --project-dir dbt docs generate
```
